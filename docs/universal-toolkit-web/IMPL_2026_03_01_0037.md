### 5. Shared Components (Shadcn/ui)

#### Cài đặt Shadcn components cần thiết

```bash
npx shadcn@latest add button sidebar breadcrumb input separator tooltip badge scroll-area sheet
```

> **Note:** Shadcn components sẽ được generate vào `components/ui/` (theo `components.json` config).
> Ngoài ra, `components.json` hiện đang map `ui` → `@/components/ui`. Ta sẽ giữ nguyên config này vì đây là convention chuẩn của Shadcn. Trong `idea-architecture.md` đề xuất `shared/components/ui/` nhưng sẽ điều chỉnh để phù hợp với Shadcn config sẵn có, tránh conflict.

## Verification Plan

1. **Build check**: Chạy `npm run build` để đảm bảo không có lỗi TypeScript hay build errors
2. **Lint check**: Chạy `npm run lint` để kiểm tra code style

### Browser Verification

1. Chạy `npm run dev` và mở browser tại `http://localhost:3000`
2. Kiểm tra:
   - Dashboard/Home page hiển thị đúng với danh sách tools
   - Sidebar menu hiển thị các tool categories và tools
   - Click vào tool → navigate đến tool page, tool được lazy load
   - Error Boundary hoạt động (cố tình throw error trong 1 tool)
   - Dark/Light theme toggle hoạt động
   - Sidebar collapse/expand hoạt động
   - Search tool trong sidebar hoạt động
   - Breadcrumb hiển thị đúng
   - 3 tools (JSON Formatter, Base64 Encoder, Color Picker) hoạt động cơ bản

> **Important:** Project hiện tại chưa có test framework (không có Jest/Vitest). Verification chủ yếu dựa vào build check và browser testing.
